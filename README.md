GitHub Pages로 올리는 가장 쉬운 방법:

1. GitHub에 새 저장소를 만듭니다. 예: `8up-legal`
2. 이 `legal/` 폴더 안 파일 4개를 새 저장소 루트에 그대로 올립니다.
3. 저장소 `Settings > Pages` 로 이동합니다.
4. `Build and deployment`에서 `Source`를 `Deploy from a branch`로 선택합니다.
5. Branch를 `main` / folder를 `/ (root)`로 선택하고 저장합니다.
6. 몇 분 뒤 `https://<github-username>.github.io/8up-legal/` 형태의 URL이 발급됩니다.
7. 실제 제출 URL은 아래처럼 사용합니다.
   - `https://<github-username>.github.io/8up-legal/privacy-policy.html`
   - `https://<github-username>.github.io/8up-legal/terms-of-service.html`

주의:

- 정책 문구가 바뀌면 GitHub Pages에 올린 파일도 같이 갱신해야 합니다.
- `8up.kr`은 현재 Firebase Hosting에 연결하려는 메인 도메인으로 쓰고 있으므로, 정책 문서는 우선 GitHub Pages 기본 URL로 운영하는 편이 가장 단순합니다.
